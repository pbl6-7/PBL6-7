# 校园活动发布平台

## 项目简介

校园活动发布平台是一个轻量型的校园活动管理Web应用，旨在为校园活动的组织和参与提供便捷的数字化解决方案。平台支持活动发布、活动报名、评论互动、收藏管理等核心功能，方便活动组织者管理活动信息，提高师生参与度。

## 功能特性

### 用户管理
- 用户注册与登录（学号/工号 + 邮箱）
- 个人信息管理
- 密码修改

### 活动管理
- 发布活动
- 编辑活动
- 删除活动
- 活动列表管理

### 活动报名
- 在线报名
- 报名信息查看
- 报名状态管理

### 互动功能
- 活动评论
- 活动收藏

## 技术栈

- 前端：响应式Web设计，支持PC端和移动端
- 后端：轻量级Web框架
- 数据库：关系型数据库

## 开发指南

### 分支说明
- `main` - 主分支，稳定版本
- `YX` - 开发分支，用于新功能开发

### 贡献流程
1. 从 `main` 分支创建新功能分支
2. 在新分支上开发并提交代码
3. 开发完成后创建 Pull Request
4. 代码审核通过后合并到主分支
