## 二、必备安装软件

### 1. Maven（必须）

**作用**：项目构建和依赖管理

**版本要求**：Maven 3.6+

**安装步骤**：

1. 下载 Maven：https://maven.apache.org/download.cgi
2. 解压到：`C:\apache-maven-3.9.6`
3. 配置环境变量：
   - 新建系统变量 `MAVEN_HOME` = `C:\apache-maven-3.9.6`
   - 在 Path 中添加：`%MAVEN_HOME%\bin`
4. 验证安装：`mvn -version`

---

### 2. MySQL（必须）

**作用**：关系型数据库存储数据

**版本要求**：MySQL 8.0+

**安装步骤**：

1. 下载 MySQL：https://dev.mysql.com/downloads/installer/
2. 安装时选择：
   - Server only（仅安装服务器）
   - 端口：3306
   - 密码：root
3. 验证安装：`mysql -u root -proot`

**创建数据库**：
```sql
CREATE DATABASE campus_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

## 四、快速验证

安装完成后，在项目目录执行：

```bash
# 1. 验证环境
java -version
mvn -version
mysql -u root -proot -e "SELECT 1"

# 2. 创建数据库
mysql -u root -proot -e "CREATE DATABASE IF NOT EXISTS campus_db;"

# 3. 编译运行
mvn clean compile
mvn spring-boot:run

# 4. 访问
# Swagger UI: http://localhost:8080/swagger-ui/
```

## 五、常见问题

### Q: Maven 下载依赖太慢？
A: 配置国内镜像源，编辑 `maven/conf/settings.xml`

### Q: MySQL 连接不上？
A: 检查服务是否启动：`services.msc` → MySQL80
